분석 파이프라인 (Layer 기반)

1. 패키지명 추출 Extension: MutationObserver
→ 코드블록 / pip install
→ 백틱 / import / 인기 패키지 매칭
CLI: AST + Regex 하이브리드
2. Layer 1 — 메타데이터 PyPI/npm API 조회
• 미등록 → +70~80점
• 신규 등록 (7일) → +25점
• 합성 패턴 → +20점
• 유사 이름 → +20점
3. Layer 2 — 소스 분석 등록된 패키지만 실행
• 아카이브 메모리 스트리밍 (5MB 제한)
• 악성 패턴 9종 탐지
• base64+exec, credential theft 등
• Shannon 엔트로피 (난독화)
4. 최종 판정 미등록 → L1 점수만
등록됨 → L2 + 메타 보조

Automation — n8n Workflow (:5679) + Gemini API
